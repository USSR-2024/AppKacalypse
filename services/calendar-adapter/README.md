# calendar-adapter (опционально, FUTURE)

Зарезервированное место под двусторонний/push-режим календаря (CalDAV-write).
**В MVP не реализуется** — см. `docs/CALENDAR.md`.

Текущая модель: Vikunja = точка правды, календари пользователей подписываются на
ICS/CalDAV-фид Vikunja (одностороннее, без этого адаптера).

Появится здесь, если понадобится писать события прямо в личные календари или
синхронизировать создание событий из календаря обратно в Vikunja:
- `app/main.py` — FastAPI
- `app/providers/{caldav,google,nextcloud}_provider.py`
