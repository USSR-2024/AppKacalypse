## Что это
Собственный AI-диспетчер задач/заметок/напоминаний/календаря: пользователь пишет обычным языком в Telegram/MAX/email → локальный LLM (Qwen) извлекает структуру → создаются задачи в Vikunja и события в календаре. **Одно-серверная архитектура, только локальный Qwen, без внешних API.**
