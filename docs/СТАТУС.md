## Что это
Собственный AI-диспетчер задач/заметок/напоминаний/календаря: пользователь пишет обычным языком в Telegram/email → локальный LLM (Qwen) извлекает структуру → создаются задачи в Vikunja. **Одно-серверная архитектура, только локальный Qwen, без внешних API.**
